- This REST API is the backend of my Responsive React Chat App.

- MVC architecture
- Web Socket
- Express.js
- Bcrypt
- JWT
- Mongoose

# Routes

- /auth/login
- /auth/register
- /auth/profile
- /auth/logout

- /users
